The rapid pace of product design can make it easy to lose track of color and component variants, especially in large-scale environments.

Beyond challenges in the hand-off process and production, maintaining consistency even in Figma design files for multi-product environments can be difficult.

Chassis - CSS helps centralize all design decisions using design tokens, which can be seamlessly pushed to Figma prototypes and code repositories for multiple products.

With a fully tokenized Figma component library, designers can quickly create prototypes for different brands and push updated design decisions directly to code repositories.

Designers have full control over settings such as sizing, spacing, icon colors, border radii, font sizes, and detailed typography (including letter-spacing). These settings can be applied to Figma prototypes or product releases with ease.

## Let's Get Started

### From Figma to Code

Let’s apply these Figma font specifications using Chassis - CSS text utilities.

<a href="#imageModal" data-cx-toggle="modal"

These Figma properties can be applied as follows:

<Example code={`<p>font/text/medium/normal</p>
<p class="font-large font-mass">font/text/large/mass</p>
<p class="font-display font-strong">font/display/medium/strong</p>`} />

The same approach applies to other elements.

<a href="#imageModal" data-cx-toggle="modal"
  data-image-src="/assets/docs-figma-frame-specs.png"
  data-image-srcset="/assets/docs-figma-frame-specs.png.png, /assets/docs-figma-frame-specs@2x.png 2x"
  data-image-alt="Figma frame layer properties">
  <img id="image" class="d-block mb-medium image-fluid rounded" srcset="/assets/docs-figma-frame-specs.png.png, /assets/docs-figma-frame-specs@2x.png 2x" src="/assets/docs-figma-frame-specs.png" alt="Figma frame layer properties" /></a>

Let’s apply these frame specifications in left-to-right order:

<Example code={`<div style="width: 128px; height: 128px;"
    class="rounded-xlarge border border-2xlarge
      py-small px-medium gap-2xsmall
      default-bg-main primary-border-subtle
      shadow-40 shadow-primary">
    I look just like in Figma!
  </div>`} />

It’s that simple. Designers now have full control over these values, which can be updated without developer intervention for individual brands, products, or even CSS themes.

### Connecting Figma with Chassis - CSS

Chassis - CSS's fully tokenized Figma component library allows designers to preview design changes across multiple apps and brands in Figma. These changes can then be injected directly into developers' code as transformed class files for each platform.
