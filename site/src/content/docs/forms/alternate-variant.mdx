## How it works

Chassis CSS provides an alternate styling variant for form elements that uses a different color scheme than the default styling. This provides flexibility in design while maintaining consistent styling across light and dark themes. A key advantage of this variant is that form elements with the `.form-alternate` class retain their appearance when switching between themes, providing visual continuity. This is particularly useful in situations where you need.

The `.form-alternate` class can be applied at different levels:
- On an entire `<form>` element to cascade styling to all child form controls
- On specific container elements like `.input-group` or form groups
- Directly on individual form controls (`.form-control`, `.form-select`, etc.)

When applied to container elements, the alternate styling automatically applies to all form controls within that container through CSS inheritance, eliminating the need to add the class to each individual form element.

## Container Elements

Apply the `.form-alternate` class to various container elements to automatically style all descendant form elements with the alternate styling.

### Form Element

Adding the `.form-alternate` class to the `<form>` will affect all form controls within it.

<Example code={`<form class="form-alternate">
    <div class="mb-medium">
      <label for="exampleFormControlInput1" class="form-label">Email address</label>
      <input type="email" class="form-control" id="exampleFormControlInput1" placeholder="name@example.com">
    </div>
    <div class="mb-medium">
      <label for="exampleFormControlTextarea1" class="form-label">Example textarea</label>
      <textarea class="form-control" id="exampleFormControlTextarea1" rows="3"></textarea>
    </div>
    <button type="submit" class="button primary">Submit</button>
  </form>`} />

### Input Groups

Input groups can also be styledby applying the `.form-alternate` class to the `.input-group` container.

<Example code={`<div class="row g-xlarge">
    <div class="col">
      <label for="default-input-group" class="form-label">Default input group</label>
      <div class="input-group mb-medium">
        <span class="input-addon">@</span>
        <input type="text" class="form-control" id="default-input-group" placeholder="Username">
      </div>
    </div>

    <div class="col">
      <label for="alternate-input-group" class="form-label">Alternate input group</label>
      <div class="input-group mb-medium form-alternate">
        <span class="input-addon">@</span>
        <input type="text" class="form-control" id="alternate-input-group" placeholder="Username">
      </div>
    </div>
  </div>
  `} />

### Selective Container

You can apply `.form-alternate` to any container element to style specific sections of your form.

<Example code={`<form>
    <div class="form-default row g-xlarge">
      <div class="mb-medium col-medium-6">
        <label for="firstName" class="form-label">First name</label>
        <input type="text" class="form-control" id="firstName">
      </div>
      <div class="mb-medium col-medium-6">
        <label for="lastName" class="form-label">Last name</label>
        <input type="text" class="form-control" id="lastName">
      </div>
    </div>
    <!-- Alternate section -->
    <div class="form-alternate row g-xlarge">
      <div class="mb-medium col-medium-6">

    <button type="submit" class="button primary">Submit</button>
  </form>`} />

## Floating Labels

Floating labels work with alternate form styling as well. Simply add the `.form-alternate` class to the form or to the parent `.form-floating` container. This provides a consistent alternate appearance while maintaining the floating behavior.

<Example showMarkup={false} code={`<div class="row g-xlarge">
    <div class="col-medium-6">
      <h5>Default Floating Labels</h5>
      <div class="form-floating mb-medium">
        <input type="email" class="form-control" id="floatingDefaultInput" placeholder="name@example.com">
        <label for="floatingDefaultInput">Email address</label>
      </div>
      <div class="form-floating mb-medium">
        <select class="form-select" id="floatingDefaultSelect">
          <option selected>Select an option</option>
          <option value="1">Option 1</option>
          <option value="2">Option 2</option>
        </select>
        <label for="floatingDefaultSelect">Select menu</label>
      </div>
      <div class="form-floating">
        <textarea class="form-control" placeholder="Leave a comment here" id="floatingDefaultTextarea" style="height: 100px"></textarea>
        <label for="floatingDefaultTextarea">Comments</label>
      </div>
    </div>

    <div class="col-medium-6">
      <h5>Alternate Floating Labels</h5>
      <div class="form-floating form-alternate mb-medium">
        <input type="email" class="form-control" id="floatingAlternateInput" placeholder="name@example.com">
        <label for="floatingAlternateInput">Email address</label>
      </div>
      <div class="form-floating form-alternate mb-medium">
        <select class="form-select" id="floatingAlternateSelect">
